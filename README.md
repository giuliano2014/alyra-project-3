## Launch the Dapp

First ensure you are in the right directory.
You must complete these 3 steps to get the project working locally.

Start the locale node.

```sh
$ cd client
$ ganache
```

Deploy the contracts.

```sh
$ cd client
$ truffle migrate --reset
```

Start the react dev server.

```sh
$ cd client
$ npm start
```
